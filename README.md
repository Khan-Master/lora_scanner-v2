# lora_scanner-v2
1  Copy checkpoint_scanner v2 to the folder you want to work in.
2  Double-click the folder to open it.
3  In the folder’s address bar, type cmd and press Enter to open the Command Prompt.
4  In the Command Prompt, install the required package by typing:
5  pip install safetensors
6  and press Enter.
7  After the installation is complete, double-click the .py file to run checkpoint_scanner v2.
enjoy!!!
